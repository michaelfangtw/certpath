# CertPath

> Gamified certification learning platform — adaptive daily plans, spaced repetition, and AI conversation practice.

Currently focused on TOEIC Golden Certificate (860+), designed to expand to other certifications.

## Stack

- React + Vite + TypeScript
- Supabase Auth (Google OAuth)
- IndexedDB via Dexie.js
- Vercel (Phase A) → Google Cloud Run (Phase B)

## Prototype

Open `project/TOEIC Golden Certs.html` in a browser to view the UI prototype.
